# titanic_survival_prediction
Модель предсказания выживаемости пассажиров «Титаника» – личный проект 2026
•	Цель проекта: Разработка модели машинного обучения для бинарной классификации (выжил/погиб) пассажиров на основе социально-демографических признаков.
Ключевые этапы реализации:
•	Проведен разведочный анализ данных, санитарная обработка и реализована экстрация признаков.
•	В качестве baseline-модели использована логистическая регрессия (accuracy = 81.01%)
•	Для учета нелинейных зависимостей осуществлен переход к ансамблевым методам. Выбран метод случайного леса, показавший лучший результат на валидационной выборке — 82.68%.
•	Средняя точность на кросс-валидации составила 82.94% при стандартном отклонении 0.0187, что подтверждает стабильность модели на новых данных.
