# Titanic Survival Prediction

Данный проект посвящен решению классической задачи бинарной классификации на платформе Kaggle. Цель — предсказать выживаемость пассажиров на основе их данных.

## Стек технологий
* Python, Pandas, Scikit-learn, Numpy, Seaborn, Matplotlib.

## Ключевые поинты проекта
* Проведен разведочный анализ данных, санитарная обработка и реализована экстрация признаков.
* В качестве baseline-модели использована логистическая регрессия (accuracy = 81.01%).
* Для учета нелинейных зависимостей осуществлен переход к ансамблевым методам. Выбран метод случайного леса, показавший лучший результат на валидационной выборке.
* Средняя точность на кросс-валидации составила 82.94% при стандартном отклонении 0.0187, что подтверждает стабильность модели на новых данных.

## Результаты
* **Accuracy на валидации:** 82.68%
* **Mean CV Accuracy:** 82.94%
* **Kaggle Public Score:** 0.77990

## Как запустить
1. Склонируйте репозиторий.
2. Установите зависимости: `pip install pandas scikit-learn matplotlib seaborn`.
3. Откройте ноутбук `titanic.ipynb`.
